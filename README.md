## Project structure

Portfolio content lives in `src/components/portfolio/`:

| File (UI label) | Source |
|-----------------|--------|
| `about.dart` | `sections.tsx` + `data.ts` (`PROFILE`) |
| `skills.dart` | `data.ts` (`SKILLS`) |
| `experience.dart` | `data.ts` (`EXPERIENCE`) |
| `projects.dart` | `data.ts` (`PROJECTS`) |
| `education.dart` | `data.ts` (`EDUCATION`) |
| `contact.dart` | `data.ts` (`PROFILE`) |

Edit **`src/components/portfolio/data.ts`** to update your information.
